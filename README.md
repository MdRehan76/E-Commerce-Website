# Amazon Clone (Frontend)

A simple Amazon homepage clone built with HTML and CSS.

## Tech Stack

- HTML5
- CSS3

## Project Structure

- `index.html` - main page markup
- `StyleAmazon.css` - styling for the page
- `photos/` - image assets used in the UI
